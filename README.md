# 80s-Chatbot

Welcome to **80s-Chatbot** — a retro style chatbot built using the Gemini API. It’s inspired by the neon lights, cassette tapes, big hair, and synthwave vibes of the 1980s. The chatbot responds with a tone and personality reflecting that era.

---

## Features

- Replies using an 80’s-themed style / slang / atmosphere  
- Uses [Gemini API] for language generation  
- Web-UI built with React + Vite (frontend)  
- Easy to install and run locally  

---

## Installation & Setup

Follow these steps to clone, configure, and run the chatbot locally:

```bash
# 1. Clone the repository
git clone https://github.com/Muksit86/80s-Chatbot.git

# 2. Change directory into the project
cd 80s-Chatbot

# 3. Install dependencies
npm install   # for frontend React / Vite

# 4. Set up your Gemini API credentials
#    Create a file like .env in the project root, for example:
#    GEMINI_API_KEY=your_api_key_here

# 5. Start the development server
npm run dev

# 6. Open your browser and go to
#    http://localhost:5017 (or whatever port is shown in console)
